# 3. UI Wireframe Description

## Visual Direction

- Design inspiration: Stripe, Notion, Linear, Bloombyte
- Interface principles:
  - clean UI
  - soft gradients
  - rounded cards
  - modern dashboards
  - role-specific clarity
  - high readability
  - lightweight enterprise feel

## Marketing Website Wireframe

### Home

- Top nav with compact brand block and CTA
- Large left-aligned hero copy
- Right-side floating dashboard mockup
- Problem cards in a three-column grid
- Module cards in a four-column grid
- Benefits split section with dark callout and supporting copy
- Footer with product and resource links

### Product Pages

- Page hero with SEO badge, headline, subheadline
- Feature card grid
- Supporting side-by-side explanatory panels
- Strong end CTA

### Mobile UI Style

- Minimal
- Student-friendly
- Fewer dense tables
- More cards and timelines
- Fast-loading views
- Action-oriented notifications

## UX Notes

- Differentiate faculty and mentor roles clearly because institutions often split these responsibilities
- Show alerts early instead of burying them in reports
- Use color and status tags for attendance risk, academic risk, and pending actions
- Keep data-entry workflows fast for faculty
- Keep review workflows insight-rich for admins and mentors

## Dashboard Information Architecture

### Admin Dashboard

- Header: campus selector, notification tray, search, profile
- KPI row:
  - Total students
  - Attendance percentage
  - Defaulters
  - Mentor groups
  - Departments active
- Main panels:
  - Department analytics
  - Attendance heatmap
  - Academic performance insights
  - Pending approvals
  - Mentor allocation summary
- Actions:
  - Create departments
  - Create mentors
  - Create courses
  - Upload student master
  - Assign mentors
  - Configure rules
  - Generate reports

### Faculty Dashboard

- Header: assigned courses, current semester, quick attendance button
- Panels:
  - Today’s classes
  - Attendance pending
  - Marks entry queue
  - Course materials
  - Student analytics by subject
- Actions:
  - Mark attendance
  - Upload marks
  - Upload materials
  - View class performance

### Mentor Dashboard

- Header: assigned mentor groups, alerts, search students
- Panels:
  - Assigned students
  - Attendance alerts
  - Meeting calendar
  - Student performance tracker
  - Communication center
- Actions:
  - Record mentor meetings
  - Add notes
  - Escalate concern
  - Send message

### Student Dashboard

- Header: semester snapshot, alert center, profile menu
- Panels:
  - Attendance summary
  - Marks and GPA
  - Timetable
  - Announcements
  - Mentor messages
  - Assignments
- Actions:
  - View detailed attendance
  - Download marksheet
  - Open timetable
  - Read notice

# 8. Product Roadmap

## Phase 1: Foundation MVP

- Institution setup and tenant model
- User authentication and role-based access
- Admin dashboard
- Student management
- Department and course management
- Faculty attendance marking
- Basic student dashboard
- Announcements and notifications

## Phase 2: Academic Operations

- Subject management
- Enrollment workflows
- Internal marks entry
- Examination setup
- Result calculation
- Report generation
- Attendance analytics and defaulter alerts

## Phase 3: Mentorship Layer

- Mentor dashboard
- Mentor allocation logic
- Mentor groups
- Mentor meeting logs
- Intervention and follow-up system
- Mentor effectiveness analytics

## Phase 4: Intelligence and Scale

- Student risk scoring
- Predictive attendance and performance alerts
- Executive analytics
- Multi-campus support
- Advanced export and accreditation reports
- SIS, LMS, and SSO integrations

## Phase 5: Mobile Experience

- React Native student app
- Attendance and marks views
- Timetable and assignments
- Mentor messaging
- Push notifications
